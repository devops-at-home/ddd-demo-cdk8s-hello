# ddd-demo-cdk8s-hello

AWS CDK8s for generating 'Hello World' workload manifests - demo for DDD Perth 2021

## Requirements

- NodeJS 14, NPM, and Yarn installed

## Commands

### Build and run tests

```
yarn install
yarn build
```

### Update tests

```
yarn test -u
```

### Deploy stack

### Upgrades

```
yarn run import
yarn run upgrade
```
